# IOT_Project
Concevoir une application mobile hybride simulant un scénario IoT (Internet of Things) ou AIoT (Artificial Intelligence of Things).  L’application doit interagir avec des données simulées de capteurs (température, humidité, mouvement, consommation d’énergie, etc.) et stocker ces données localement à l’aide d’une base de données SQLite (optionnelle)
